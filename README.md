# ALISEditor

## Structure

ALIS Editor is a rich text editor based on tree structure by JSON.

## How to development

### Clone project

```bash
$ git clone git@github.com:AlisProject/alis-editor.git
$ cd
```

### Run development server

w/ Vue CLI

```bash
$ yarn # Install deps
$ yarn serve
```

### Install deps

w/ Vue CLI UI

```bash
$ yarn vue ui # Open Vue CLI v3 UI server
```

## Testing

w/ jest

```bash
$ yarn test
```

## Lint and format

w/ prettier

```bash
$ yarn format
```
