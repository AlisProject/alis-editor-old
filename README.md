# ALISEditor

## Structure

ALIS Editor is a rich text editor based on tree structure by JSON.

Tree structure
```
ALIEditor
├── 1_Paragraph
│   └── 1_Text
├── 2_Paragraph
│   ├── 1_Text
│   ├── 2_Link
│   │   └── 1_Text
│   └── 3_Text
├── 3_Rule
├── 4_Title
│   └── 1_Text
├── 5_Image
└── 6_Link
    └── 1_Image
```
